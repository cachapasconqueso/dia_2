# Desarrollo de software
## El desarrollo de software es una de las crónicas más fascinantes de la humanidad: Es cómo pasamos de mover cables manualmente a construir mundos virtuales.

La programación consistía en configurar físicamente interruptores y conectar cables en paneles.

- Años 40-50 (Hardware Puro): No había archivos. Programar era mover cables y palancas físicamente (como en la ENIAC). El código nació como instrucciones binarias (0 y 1).

- Años 50-60 (Abstracción): Se inventan los compiladores. Ahora los humanos pueden escribir palabras (FORTRAN, COBOL) que la máquina traduce. Usaban tarjetas perforadas.

- Años 70 (Estandarización): Nace el Lenguaje C, la base de casi todo lo que usamos hoy. Se empieza a tratar el software como una ingeniería seria para evitar el caos.

- Años 80-90 (Globalización): Aparecen las ventanas e íconos (GUI) y la Programación Orientada a Objetos (Java, C++). El software llega a las casas en PCs.

- Años 2000 (Agilidad y Red): El software se vuelve Open Source (Linux) y se descarga de Internet. Se crean los Métodos Ágiles para programar más rápido y en equipo.

- 2010 - Hoy (Nube e IA): El software ya no se instala, está en la Nube (SaaS). Ahora, las IA ayudan a los humanos a escribir el propio código.
